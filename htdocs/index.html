LS4, large-scale simple storage system for multimedia contents

LS4 is a distributed storage system suitable for storing data like photos, musics or movies with attributes. 

$ gem install ls4
$ ls4-standalone -s ./data &
$ ls4cmd --help

Data Model

LS4 stores a set of objects identified by a key. Each object consists of data and attributes. And each object can have multiple versions. You can get back old version of the object unless you surely delete it. The versions are identified by name or creation time.
